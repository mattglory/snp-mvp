# Screenshots Folder

## Instructions

Please add your 4 SNP product screenshots to this folder:

1. **dashboard.png** - Main dashboard showing TVL and APY stats
2. **vault_selector.png** - Three vault selection cards (Conservative, Balanced, Growth)
3. **comparison_table.png** - Vault comparison table with APY and risk levels
4. **portfolio_view.png** - Portfolio management interface

## How to Get Screenshots

1. Run the frontend:
   ```bash
   cd C:\snp-mvp\frontend
   npm start
   ```

2. Open browser to http://localhost:3000

3. Use Windows Snipping Tool (Win+Shift+S) or Screenshot function to capture each section

4. Save as PNG files in this folder with the names above

## Screenshot Tips

- Use full browser width for best quality
- Capture clean sections without browser chrome
- Ensure dark theme is visible
- Show all three vault options in vault selector
