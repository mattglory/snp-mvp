## Troubleshooting

**Can't find screenshots?**
1. Press `Win + S`
2. Search for: `*.png`
3. Filter by "Date modified" = Today
4. Look for files named dashboard, vault_selector, comparison_table, portfolio_view

**Google Drive upload too slow?**
- Upload files individually if the folder upload fails
- Make sure you have good internet connection
- The total size should be small (< 5MB)

**Calendly confusion?**
- Just set up a simple "30 minute meeting" event
- Set it to your timezone
- Add some availability slots in the next 2-4 weeks
- The link is what matters, not the settings
